About

社内SEとして働きながら、副業として「データ分析 × 業務改善」を仕事にすべく勉強中です。

いま取り組んでいること

  • Kaggleコンペ参戦 × Claude Code高速PDCA — Claude Codeをペアに据え、仮説立案 → 実装 → 結果読み解き → 次の一手 のサイクルを高速回転。「思いつきで特徴量を足す」のではなく「仮説を立てて検証した結果こうだった」を積み上げる分析の型を、AI駆動で実戦検証しています。
  • Claude Code の個人運用環境づくり — 「毎回AIに前提を入れ直すのがめんどくさい → ならばナレッジを蓄積して自分専用のアドバイザーを育てる」思想で、メモリ・Hook・スキルを組み合わせた個人運用インフラを日々アップデート。ここで磨いた環境がKaggleのPDCAの土台になっています。

副業として繋げたい先

  • データ分析を軸にした業務改善の受託・コンサル
  • Claude Code 個人運用ノウハウの企業導入支援・勉強会登壇・記事発信
  • 上記2つを組み合わせた「企業のAI活用支援」を小規模案件から積み上げる
